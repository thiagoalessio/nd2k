# nd2k

Converts NovaDAX CSVs reports to formats accepted by Koinly.

[![CI][ci_badge]][ci]

## Requirements

* Python 3.10 or later

## Installation and Usage

	$ pip3 install git+https://github.com/thiagoalessio/nd2k.git
	$ nd2k novadax-file.csv
